# Open Source Survey Results - Cookiecutter Template

A Cookiecutter template for creating MyST-based survey result websites.

## Usage
```bash
pip install cookiecutter
cookiecutter https://github.com/UW-Madison-DSI/open_source_survey_results_myst
```

Answer the prompts to customize for your institution.

## What You'll Get

- MyST Markdown-based Jupyter Book
- Customizable branding (colors, logos, institution name)
- GitHub Actions workflow for automatic deployment
- Python-based data analysis and visualization
- Responsive design with custom footer

## Requirements

- Python 3.11+
- Jupyter Book <2.0
- pandas, plotly, matplotlib
