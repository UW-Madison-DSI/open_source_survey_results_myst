# {{ cookiecutter.project_name }}

Survey results and analysis for {{ cookiecutter.institution_name }}.

## Quick Start
```bash
cd survey_results_docs/docs
pip install 'jupyter-book<2.0' pandas plotly matplotlib
jupyter-book build .
open _build/html/index.html
```

## Customization

- Update `_config.yml` for site settings
- Update `setup.py` for colors and branding
- Replace `data/survey_data.csv` with your survey data
- Customize markdown files in `docs/` directory

For more details, see the reproduction section in the built site.
